== Description ==

This plugin adds a widget that can show the statistics of a betting blog (e.g. the amount of units, the number of bets so far, the percentage yield, etc.). Currently the user needs to update the statistics manually using a form in the admin area.

== Installation ==
In the zip file you will find a directory called: wp_betting_stats.

1. Upload the directory to the /wp-content/plugins/ directory of your wordpress installation.
2. Activate the plugin throught the Plugins menu in wordpress.

== How to Use ==
A widget named: "WP Betting Statistics" will be available in the "Widgets" subsection of the "Appearance" section of the wordpress admin area. You can add the widget to your theme.

Under the "Settings" section of your wordpress admin area, a submenu called: "WP Betting Stats" should appear.  There you can add the values you want to the relevant fields. Those are the values the visitors of your site will see in the area of your theme that you placed the widget.
